\doxysubsection{Detailed Description}
Provides interface to GIOS REST API for air quality data. 

This class handles network communication and data processing for\+:
\begin{DoxyItemize}
\item Retrieving list of monitoring stations
\item Fetching station details
\item Getting sensor measurements 
\end{DoxyItemize}
