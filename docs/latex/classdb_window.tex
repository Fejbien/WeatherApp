\doxysubsection{Detailed Description}
Provides a GUI interface for browsing and loading saved sensor data. 

The window displays\+:
\begin{DoxyItemize}
\item List of available cities (as directories)
\item JSON files for selected city
\item Allows loading data back into the main application 
\end{DoxyItemize}
